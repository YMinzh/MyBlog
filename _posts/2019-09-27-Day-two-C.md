## vim工具的使用与C语言的学习

###### vim的基本指令

| 指令          | 作用                                 |
| ------------- | ------------------------------------ |
| Esc           | 进入命令模式                         |
| a             | 插入模式,插入在光标后面              |
| i             | 插入模式,插入在光标前面              |
| w             | 保存                                 |
| q             | 退出                                 |
| shift+a       | 光标移动至行尾                       |
| shift+i       | 光标移动至行首                       |
| o             | 光标移动至下一行                     |
| shift+o       | 光标移动至上一行                     |
| dd            | 删除一行(剪切  前面加数字可多行操作) |
| x             | 删除一个字母                         |
| sp   新文件名 | 创建一个新的文件                     |
| set  nu       | 打开行号                             |
| p             | 粘贴                                 |
| ctrl+w+上/下  | 在两个文件间转换                     |
| wqa           | 保存并退出所有文件                   |

###### c语言的标准流

1. printf("please intput a value");  ==fprintf(stdout",please input a value");
2. scanf("%d",&a);  ==fscanf(stdin,"%d",&a);
3. fprintf(stderr,"the value must >0");  return 1;  //返回1是为了让操作系统知道出错，并返回错误输出流

###### 重定向

|      | 输入 | 输出 |
